# Project Summary
The **Lightweight Image Classifier** is an open-source image classification framework built using PyTorch. It provides a flexible and efficient solution for training and deploying various lightweight models, including ResNet, MobileNet, and ShuffleNet. The project aims to facilitate quick and accurate image classification for developers and researchers, supporting both CPU and GPU training environments.

# Project Module Description
The project consists of several functional modules:
- **configs**: Configuration files for model parameters and training settings.
- **datasets**: Custom dataset classes and data augmentation techniques.
- **models**: Implementations of various lightweight models.
- **scripts**: Training, validation, inference, and ONNX export scripts.
- **tests**: Unit tests for models and datasets.
- **utils**: Utility functions for logging, metrics, checkpoint management, and visualization.
- **docker**: Docker configuration for easy deployment.

# Usage
To install dependencies and run the project, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/lightweight-image-classifier.git
   cd lightweight-image-classifier
   ```

2. **Set up a virtual environment**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use: venv\Scripts\activate
   ```

3. **Install required packages**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Train the model**:
   ```bash
   python scripts/train.py --model resnet18 --data_dir ./data --num_classes 10
   ```

5. **Validate the model**:
   ```bash
   python scripts/validate.py --model resnet18 --checkpoint ./checkpoints/best_model.pth --data_dir ./data --num_classes 10
   ```

6. **Run inference**:
   ```bash
   python scripts/inference.py --model resnet18 --checkpoint ./checkpoints/best_model.pth --image ./test.jpg --num_classes 10 --topk 5
   ```

7. **Export to ONNX**:
   ```bash
   python scripts/export_onnx.py --model resnet18 --checkpoint ./checkpoints/best_model.pth --num_classes 10 --output model.onnx
   ```

8. **Run Docker**:
   ```bash
   docker-compose up gpu-trainer
   ```
